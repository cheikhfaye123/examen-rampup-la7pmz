Examen Ramp Up

Objetivo:

¡Enhorabuena por llegar hasta aquí! Ahora queda una prueba más para pasar el Ramp Up. Demuestra todo lo que has aprendido resolviendo estos ejercicios:

HTML Y CSS

1. Diseña un formulario de contacto en HTML con campos para:

  • Nombre
  • Email
  • Asunto del mensaje
  • Mensaje
  • Botón para enviar
  • Botón para borrar todo

Los botones aumentarán de tamaño al pasar el ratón sobre ellos y tendrán que cambiar de color.

El campo email tiene que ser obligatorio. Cuando pulsemos el botón enviar si está vacio debe salir algo que diga que tiene que rellenarlo (piensa en algún atributo)

Debes darle estilos al formulario usando flex, usar transiciones y transformaciones.


Javascript

1. Escribe un programa que muestre los números del 1 al 100 en la consola.

  • Para los múltiplos de 3, en lugar de mostrar el número, muestra la palabra "Fizz".
  • Para los múltiplos de 5, muestra la palabra "Buzz".
  • Para los números que son múltiplos de ambos 3 y 5, muestra "FizzBuzz".
  • Para los números que no cumplen ninguna de las condiciones anteriores, muestra el número tal como es.

2. Escribe un programa que solicite al usuario una cantidad en metros y luego le permita convertirla a pies o pulgadas mediante 'prompt()'. Se debe mostrar el resultado de la conversión en la consola. 1 metro equivale a 3.281 pies y a 39.37 pulgadas.

Recuerda pasar el "prompt" a tipo número para hacer cálculos
